https://github.com/MordWraith/AuraTracker

GameHelper2 nearby-enemy panel: HP/ES bars, buff/debuff icons per target, per-enemy DPS and optional total DPS header, draggable layout and filters. Based on Skrip's AuraTracker (derekShaheen upstream); GameHelper2 port with DE/EN settings UI by MordWraith. Read-only overlay — no input automation.

Build from source only (no prebuilt DLL). Clone into Plugins/AuraTracker inside your GameHelper2 tree, then dotnet build GameHelper and the plugin. Enable it under GameHelper -> Plugins.

Compatible with Gordin GameHelper2 main, .NET 10 (net10.0-windows, x64). Tested on upstream main @ 4f6fd2a (June 2026); should work on community forks using the same API.

Feedback / feature requests: this thread. Crashes or install issues: #help with tag plugin.
