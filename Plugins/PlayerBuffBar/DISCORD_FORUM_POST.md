## Paragraph post (copy-paste)

```
https://github.com/MordWraith/PlayerBuffBar

GameHelper2 player buff overlay for PoE2: up to four independent buff bars, each with its own watchlist and screen position (HP anchor or drag dummy). Separate resource row for power/frenzy/endurance charges (poe2db icons) and rage. Icons auto-download from poe2db.tw; durations and stack counts; inactive watchlist preview. Includes skill→buff aliases (e.g. refutation → runic_fortress).

Written by MordWraith — read-only overlay, no input automation.

Build from source only (no prebuilt DLL): clone into Plugins/PlayerBuffBar inside your GameHelper2 tree, then dotnet build GameHelper and the plugin. Enable under GameHelper -> Plugins. Also in my experimental Gamehelper plugin catalog (optional install / update).

Compatible with Gordin GameHelper2 main (.NET 10, net10.0-windows x64) and community forks on the same plugin API.

Tip: use "Dump my buffs" in settings to save exact in-game buff key names while effects are active.

Feedback and feature requests: this thread. Crashes or GameHelper install issues: #help with tag plugin.

Community third-party plugin — not affiliated with Grinding Gear Games. Use at your own risk; only build from the linked source.
```
